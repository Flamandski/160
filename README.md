# Training Planner

Автор: [Кульдюков Максим]

Дата: [25.04.2026]

## Описание

Приложение "Training Planner" позволяет вести дневник тренировок: добавлять записи с датой, типом, временем, фильтровать их и сохранять данные в формате JSON.

## Функциональность

* Добавление записей о тренировках.
* Просмотр всех записей в таблице.
* Фильтрация по дате.
* Сохранение и загрузка данных из JSON‑файла.
* Валидация ввода данных.

## Требования
- Python 3.6 или выше
- Стандартная библиотека Python (tkinter, json, os)
